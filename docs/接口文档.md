# 接口文档

> 学科知识整合智能体 — API 接口定义

本系统使用 Gradio 作为前端框架，以下为后端模块的核心函数接口。

## 1. 文件解析

```python
from backend.parser.parser import parse_textbook

# 输入: 文件路径 + 教材 ID
# 输出: TextbookInfo 对象
book = parse_textbook("/path/to/book.pdf", "book_001")
# book.to_dict() -> {
#   "textbook_id": "book_001",
#   "title": "生理学",
#   "total_pages": 520,
#   "total_chars": 385000,
#   "chapters": [...]
# }
```

## 2. 知识图谱构建

```python
from backend.knowledge.extractor import build_knowledge_graph, graph_to_echarts

# 输入: 教材 dict
# 输出: 图谱 dict {nodes, edges, stats}
graph = build_knowledge_graph(book.to_dict())

# 转为 ECharts 格式
echarts = graph_to_echarts(graph)
```

## 3. 跨教材整合

```python
from backend.integration.integrator import integrate_graphs

# 输入: 图谱列表 + 总字数
# 输出: IntegrationResult
result = integrate_graphs(graphs, total_chars, target_ratio=0.30)
# result.compression_ratio  # 压缩比
# result.decisions           # 整合决策列表
```

## 4. RAG 查询

```python
from backend.rag.rag_pipeline import TextChunker, VectorIndex, rag_query

# 建立索引
chunker = TextChunker(chunk_size=600, overlap=80)
chunks = chunker.chunk_textbook(book.to_dict())
index = VectorIndex()
index.index(chunks)

# 查询
response = rag_query("什么是动作电位？", index)
# response.answer        # 回答文本
# response.citations     # 引用来源列表
```
